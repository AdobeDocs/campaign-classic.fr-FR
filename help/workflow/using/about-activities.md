# Activités de workflows{#wf-activities}



Cette section décrit toutes les activités disponibles. Selon le nœud ou le contexte dans lequel le workflow est créé/modifié, les activités disponibles peuvent varier. Par exemple, les workflows créés dans une campagne comportent des activités de diffusion spécifiques à un canal.

Les activités de workflow sont regroupées par catégorie. Quatre onglets sont disponibles en même temps.

Dans les workflows des opérations, l’onglet **[!UICONTROL Evénements]** est remplacé par l’onglet **[!UICONTROL Diffusions]** . Les activités de cet onglet sont détaillées dans la section [Activités d’action](about-action-activities.md).
