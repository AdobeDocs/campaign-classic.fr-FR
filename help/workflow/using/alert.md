# Alerte{#alert}



Une activité **Alerte** envoie un message à un groupe d&#39;opérateurs. Elle fonctionne de la même manière qu’une activité d’approbation, mais aucune réponse n’est attendue dans ce cas.

![](assets/edit_alerte.png)

Une alerte n’est pas persistante et n’est donc pas visible depuis la console. Les opérateurs et opératrices du groupe affecté doivent avoir une adresse e-mail complète pour recevoir la notification. La configuration de cette activité est similaire à celle d’une **Approbation**. Le modèle de diffusion par défaut utilisé pour alerter les opérateurs et opératrices est « alertAssignee ».
