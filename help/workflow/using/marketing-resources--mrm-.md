# Ressources marketing (MRM){#marketing-resources-mrm}

Le workflow présenté ci-dessous est installé par défaut avec le module **Ressources marketing (MRM)**. Pour plus d’informations sur ce module, consultez cette [section](../../campaign/using/designing-marketing-campaigns.md).

 <tbody>

  </tr> 
  <tr> 
   <td> <span class="uicontrol">Notification des ressources marketing</span> <br /> </td> 
   <td> <span class="uicontrol">assetMgt</span><br /> </td>

  </tr>
